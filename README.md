🚀 Project: Tic Tac Toe with AI (Java)
📌 Overview

Tic Tac Toe with AI is an interactive console-based game developed using Java. The application allows a human player to compete against an AI opponent, demonstrating core concepts of game logic, state management, and basic artificial intelligence.

🎯 Key Features
🎮 Player vs AI gameplay
🧠 Intelligent move generation (AI opponent)
📊 Real-time board updates
✅ Win, loss, and draw detection
❌ Input validation and error handling
🔁 Turn-based game mechanics
🧱 Tech Stack
Language: Java
Concepts: OOP, Arrays, Control Flow
Tools: Console (CLI-based interaction)
⚙️ Functionality

The game maintains a 3x3 board using an array structure. The player makes moves by selecting positions, while the AI generates moves automatically. After each turn, the system evaluates the board to determine if a player has won or if the game has ended in a draw.

🧠 Learning Outcomes
Implementation of game state management
Understanding of decision-making logic in AI
Hands-on experience with arrays and loops
Designing interactive CLI applications
